Although I've been using Mac OS X for many years, I've never used Xcode.  I normally use Emacs on Linux Ubuntu at home to teach my son Python.  He's using Komodo on Ubuntu.  When we are on vacation, he uses Komodo on Mac and I use Emacs on Mac.

As I was getting ready for our trip to Hawaii, I decided to update my GitHub repositories on my MacBook Pro so that my son could continue to develop his games on vacation.

Since Git doesn't install by default on the Mac, I decided to install Xcode and the Apple command line tools.

I was really surprised at the large number of low reviews and complaints about Xcode.

Since there are a number of 5 star ratings, I have to assume that the product itself is good and that there are some showstopper bugs which results in many 1 star ratings.  

I've used Apple products extensively and found the Apple hardware and OS to be of high quality.  It's surprising that the company is shipping their Xcode product with some critical bugs. 

While Apple is working on Xcode, Google is working with on Android Studio with JetBrains.  The new Android Studio is based on the IntelliJ IDEA, not Eclipse.  Currently, it's on version 0.3.2.   

The competition is stepping up their game
Android Studio Developer Preview gearing up to give Xcode some competition

Update: Friday, December 13, 2013
I wasn't able to install Xcode on my three year old MacBook Pro.  From reading the comments on the AppStore, this is a common problem.  I'm not sure if it has anything to do with my running Mac OS X Mavericks.   The good news is that I was able to install the Apple Command Line Tools with no problems.  The Apple Command Line Tools installs git and I can now move things back and forth with github easily.

I was also able to install XQuartz http://xquartz.macosforge.org with no problem and can run the applications that I need.

Despite the problem with Xcode installation, I don't think it will affect me over the Christmas holiday.   I'll try and install the tools I need on my wife's MacBook Air 11" which has the new Haswell Core i5 CPU in it with Intel HD Graphics 5000.  That should be able to run everything fine.

Gigi Read just bought an even better computer, the MacBook Air 13" with 8 gig of memory.  That's a really nice machine.  I may end up picking one up in 2014.  In the meantime, I'm plodding forward with my old 2010 MacBook Pro, a new Windows 8 notebook, and my main Linux desktop machine at home.
